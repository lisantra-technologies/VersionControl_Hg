Print the contents of a file from a specific revision

Usage:

  1.  $hg = new VersionControl_Hg('/path/to/repo');
  2.  $hg->cat('/path/to/a/file')->run();

You may also specify multiple files:

  1.  $hg = new VersionControl_Hg('/path/to/repo');
  2.  $hg->cat()->files(array('file1', 'file2'))->run();
or use a pattern:
  1.  $hg->cat()->files(array('glob'=> '**.php'))->run();

Additionaly, you may cat the contents of a file at a specific revision:

  1.  $hg = new VersionControl_Hg('/path/to/repo');
  2.  $contents = $hg->cat('file2')->revision(6)->run();
  3.  file_put_contents('file2', $content);
Not specifying a revision causes Mercurial to cat the latest version of the file.

As a convenience for the latter operation, a programmer may use the save() method:

  1.  $hg = new VersionControl_Hg('/path/to/repo');
  2.  $hg->cat('file2')->revision(6)->save('new_file_name')->to('/path')->run();

or, spell out the options in an array:

  1.  $hg->cat('file2')->save(array('name' => 'new_file_name', 'to' => '/path'))->run();

Note: if you really need multiple files, consider using the Archive command.
