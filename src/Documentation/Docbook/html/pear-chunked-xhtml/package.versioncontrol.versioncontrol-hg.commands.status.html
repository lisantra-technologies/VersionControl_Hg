Status Command

Status Command – How to get the status of a working copy

Basic Operation

Running this with no arguments will only show what Hg would show: only changes and not all files.

Output from basic use:

<?php
require_once 'VersionControl/Hg.php';
$hg = new VersionControl_Hg('/home/me/my_repository');
$status = $hg->status()->run();
?>

will return:

<?php
array(2) {
  [0]=>
  array(2) {
    ["status"]=>
    string(11) "not tracked"
    ["file"]=>
    string(10) "2010-12-27"
  }
  [1]=>
  array(2) {
    ["status"]=>
    string(11) "not tracked"
    ["file"]=>
    string(7) "New.doc"
  }
}
?>

Remember, run() can have the argument: 'verbose' to output extra status data for each file.

<?php
$status = $hg->status()->run('verbose');
?>

An additional feature of VersionControl_Hg is that by default, the status command ignores any files associated with git and subversion, which a stock Mercurial install will not do. This might be useful for when a user uses hgsubversion or hg-git.

Interpreting its Output

Default: Array

<?php
array(
    "file" => "index.php",
    "status" => "clean"
)
?>

Plain Text Codes

You may get plain text output by using the format() method chained to status().

Plain text output:

<?php
$status = $hg->status('all')->format('raw')->run();
?>

Mercurial -> VersionControl_Hg Status Codes
Mercurial VersionControl_Hg
M modified
A added
R removed
C clean
! missing
? not tracked / unversioned
I ignored

Finding Status of All Files in the Repository

Output the status of all files in the repository with the all() method chained to the command.

Status of all files in the repository:

<?php
$status = $hg->status()->all()->run();
?>

or,

<?php
$status = $hg->status('all')->run();
?>

Excluding Files

You may exclude types of files with the excluding() method chained to status(). It is specified as a glob: '**.{$FILE_EXTENSION}'

Exclude all Php files:

<?php
$status = $hg->status()->all()->excluding('**.php')->run();
?>

Finding Status of only Certain Files in the Repository

You may specify an array of filenames using files() chained to status(), which will exclude all files other than the ones specified.

Paths are relative to the repository root.

Output the staus of only index.php:

<?php
$status = $hg->status()->files(array('index.php'))->run();
?>

Finding Files Added to the Repository

Using added() will output all files added, but not yet committed to the repository.

Show added files:

<?php
$status = $hg->status()->added()->run();
?>

or,

<?php
$status = $hg->status('added')->run();
?>

Finding Files Deleted from the Repository

Use deleted() to find all files deleted from the repository, but not yet committed. You may use this with revision() or between() as well.

Show deleted files:

<?php
$status = $hg->status()->deleted()->run();
?>

or,

<?php
$status = $hg->status('deleted')->run();
?>

Finding Status of Files for a Specific Revision

The argument is typically an integer, however it still works if the argument is a valid number as a string: '4' will work, but '4things' may yield unpredictable results depending on how PHP casts it.

Show the status what changed in revision 4:

<?php
$status = $hg->status()->revision(4)->run();
?>

or,

<?php
$status = $hg->status(array('revision' => 4 ))->run();
?>

Finding Which Files Changed, were Added or Deleted Between Two Revisions

between() takes two integer arguments: the first is the revision from which it starts and the second is the revision at which the status ends. The second argument may take the keyword, 'tip'.

All files which were modified, added or deleted between revisions 3 and five:

<?php
$status = $hg->status()->between(3, 5)->run();
?>

Finding Copied Files

Show copied files:

<?php
$status = $hg->status()->copied()->run();
?>

or,

<?php
$status = $hg->status('copied')->run();
?>

Find Unmodified / Clean Files

Show unmodified files:

<?php
$status = $hg->status()->clean()->run();
?>

or,

<?php
$status = $hg->status('clean')->run();
?>

Find Modified Files

Show modified files:

<?php
$status = $hg->status()->modified()->run();
?>

or,

<?php
$status = $hg->status('modified')->run();
?>

Find Unversioned Files

Show unversioned files:

<?php
$status = $hg->status()->unknown()->run();
?>

or,

<?php
$status = $hg->status('unknown')->run();
?>
