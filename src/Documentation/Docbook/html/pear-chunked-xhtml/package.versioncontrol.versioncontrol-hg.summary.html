Introduction

A fluent interface is a compact way of achieving programmatic goals. All chained commands are terminated with run().

Getting the log of the most recent commit:

<?php
require_once 'VersionControl/Hg.php';
$hg = new VersionControl/Hg('/path/to/repo');

$hg->log()->revision('tip')->run('verbose');
?>

You may use a traditional, multiline style instead of the fluent interface. However, I have not designed VersionControl_Hg to work this way; Thus, your mileage may vary.

Long form of the above example:

<?php
require_once 'VersionControl/Hg.php';
$hg = new VersionControl/Hg('/path/to/repo');
$log = $hg->log();
$revision = $log->revision('tip');
$result = $revision->run('verbose');
?>

Possible Use Cases

For use within a build system: * to archive a repository * to produce a change log from a repository

System Requirements

  • PHP 5.2.0 or newer.

  • PEAR 1.7.4 or newer.

Dependencies

  • Mercurial

VersionControl_Hg has no other PEAR packages as dependencies.

Installation

Install simply with pear install VersionControl_Hg. It has no other PEAR packages as dependencies, therefore, --alldeps has no effect on the install.
