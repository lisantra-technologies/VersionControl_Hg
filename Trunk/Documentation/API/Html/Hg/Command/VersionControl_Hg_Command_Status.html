Class VersionControl_Hg_Command_Status

Description
Description | Vars (details) | Methods (details)

Implements interfaces:

  • VersionControl_Hg_Command_Interface

Implements the status command.

The codes used to show the status of files are: M = modified A = added R = removed C = clean ! = missing (deleted by non-hg command, but still tracked) ? = not tracked I = ignored
