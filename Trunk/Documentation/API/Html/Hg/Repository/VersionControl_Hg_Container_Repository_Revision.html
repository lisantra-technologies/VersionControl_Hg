Class VersionControl_Hg_Container_Repository_Revision

Description
Description | Vars (details) | Methods (details) Constants (details)

Composes a revision object for use in commands

Rewrite some commands to accept a VersionControl_Hg_Container_Repository_Revision

Usage:

  1.  $hg = new VersionControl_Hg('/path/to/repo');
  2.  $revision = $hg->get()->revision('22')->run();
  3.  $revision = $hg->revision('22')->run();
  4.  But, do what?
  5.  
  6.  $revision = $hg->revision('22');
  7.  $log = $hg->log($revision)->run();
  8.  Which is equivelent to
  9.  $log = $hg->log()->revision('22')->run();
  10.  or
  11.  $revision = $hg->revision('3bcf453face889')->get()->run();
