Class VersionControl_Hg_Repository

Description
Description | Methods (details) Constants (details)

Represents the Repository as an entity.

Usage:

$repo = new Repository(); $repo->load( '/path/' )

As a child of Hg.php: $hg = new Hg(); $repo = $hg->getRepository( '/path/' );

Or: $repo = hg::instance()->load('/path/'); $repo = hg::instance()->create('/path/');
