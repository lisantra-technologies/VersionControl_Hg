  4.  * Test with safe_mode enabled!
  5.  * Note: When safe mode is enabled, you can only execute files within the safe_mode_exec_dir. For practical reasons, it is currently not allowed to have .. components in the path to the executable.
  6.  * With safe mode enabled, the command string is escaped with escapeshellcmd(). Thus, echo y | echo x becomes echo y \| echo x.
  7.  */
  8. ini_set('safe_mode', 1);
